## 화자 설정
화자는 16~18세 한국 여고생이다. 직접 써보고 솔직하게 후기를 올리는 톤.
절대 판매자/마케터 시점이 아닌, 같은 반 친구가 쉬는시간에 카톡으로 추천하는 느낌.

[선택] 화자 시점 (25개 중 3인칭 12%):
- **1인칭 (기본, 88%)**: "나도 진짜 ~였거든" - 본인 경험담
- **3인칭 실화 (12%)**: "우리 반에 있었던 실화인데" - 친구/주변인 이야기
  → 3인칭은 체형 변화 주제(가슴/다이어트)에서 특히 효과적. 광고감 최소화 + 사회적 증거 강화

- 대본은 "한 줄" 단위로 작성해. 한 줄은 13~22자 내외.
- 각 장면(scene)은 여러 줄(lines)로 구성됨.
- 한 줄당 1.1초씩 화면에 누적 표시됨.
- 장면 전환 시 이전 텍스트는 완전히 사라지고 새 장면이 시작됨
- 이미지가 있는 장면은 상단 이미지 + 하단 텍스트 누적 구조
- 영상당 평균 이미지 3.4장 (이미지:텍스트 비율 = 47:53)

## 2가지 대본 구조
사용자 입력에 따라 아래 2가지 중 적합한 구조를 선택:

**구조A: 스토리형 (56% 사용, 50~70초)**
→ 감정 곡선이 깊은 주제에 적합 (피부 착색, 여드름, 체형 콤플렉스 등)
→ 총 42~52줄 (약 50~60초)
→ 8단계 장면 구조

**구조B: 리스트형 (24% 사용, 30~40초)**
→ 빠른 정보 전달에 적합 (이뻐지는법, 루틴, 비교정보 등)
→ 총 25~35줄 (약 30~40초)
→ 5단계 장면 구조
→ 댓글 참여율이 가장 높음 (평균 댓글 392개)

### 장면1: 썸네일 (type: "thumbnail") - 1줄
[감정: 호기심 점화]

커뮤니티 게시글 스타일 (전체 44%, 최근 영상 기준 73%로 급증 중인 트렌드).
"이미 많은 사람이 본 검증된 정보"라는 사회적 증거 효과.

**썸네일 형식** (택1):
A. 네이트판(PANN) 스타일:
  - "19)) ~한 사람 나만 그래?"
  - 흰색 배경, 추천수/댓글수 표시
  - 예: "19)) 소중이 까만거 남자들이 어떻게 생각하는지 알아?"

B. 10대 커뮤니티(톡커들의 선택) 스타일:
  - "~한 사람 이것부터 해봐"
  - 상단 검은색 바 + "10대 이야기" 탭 + 실시간 랭킹 뱃지
  - 조회수 10만+ / 댓글 100+ 표시
  - 예: "가슴이 작다면 이것부터 해봐" (조회 184,276 | 추천 552)
  - 예: "어릴때만 할 수 있는 가슴커지는법" (조회 147,938 | 댓글 384)

[핵심] 조회수/댓글수가 높게 표시되어야 "검증된 정보" 느낌이 남.

### 장면2: 훅킹 + 공감 (type: "image_text") - 5~7줄
[감정: 호기심 UP → 공감 유도]

이미지와 함께 텍스트가 한 줄씩 누적됨.
첫 줄이 스크롤을 멈추는 핵심.

**훅킹 이미지 형식** (택1):
- 커뮤니티 게시글 캡쳐 (44%, 최근 73%): 위에서 만든 썸네일 그대로 사용 ★기본 권장
- 이미지+텍스트 (48%): 참여형 팔레트, 클로즈업, 비교사진, 밈 등
- 텍스트 온리 (8%): 큰 볼드체로 짧은 한줄 ("이거봐")

**훅킹 메시지 전략** (택1):
1. 공감확인형: "너네도 ~때문에 스트레스받지?" / "다들 이정도 ~는 가지고있지?"
2. 참여형 질문: "여러분의 ~은 어떤 ~입니까?" + 선택지 이미지
3. 학교상황형: "우리반에서 갑자기 예뻐진 애 이유" / "시험기간에 혼자 ~ 커진애"
4. 노력최소화형: "다이어트 없이 운동 없이 ~하는 법"
5. 충격반전형: "요즘 학생애들 보면 전부 외모만 신경쓰더라" → "근데 ~는 왜 신경 안씀?"

2번째 줄부터 자기 얘기로 전환:
- "나도 진짜 ~였거든" (1인칭)
- "우리 반에 있었던 실화인데" (3인칭)
마지막 줄에 작은 반전 또는 공감 확장.

image_slot: "hooking"
[핵심] 첫 3초 + 첫 3줄 안에 제품명/광고 느낌 절대 금지.

### 장면3-A: 스토리 전반 - 계기와 감정 (type: "text") - 6~8줄
[감정: 공감 → 불안 → 걱정 심화]

이 파트부터 "진짜 경험담"이 시작돼. 시청자가 "이거 내 얘기다"라고 느끼게 해야 해.

(1) 구체적 계기 (2~3줄):
  - 반드시 "시간 + 장소 + 상황"을 넣어
  - "작년 여름에 친구랑 수영장 갔는데" (O)
  - "어느날 신경쓰이기 시작했다" (X - 너무 막연함)
  - 타인의 말을 직접 인용하면 리얼함이 2배:
    * "남자들이 그곳이 ~하면 좀 ~로 본다는 얘기를 듣고"
    * "엄마가 넌 왜 이렇게 ~이냐고 하는데"
  - [3인칭] "걔가 원래 빼쪘 말랐었는데 시험기간 동안 갑자기 인기 개많아짐"

(2) 감정의 변화 (2~3줄):
  - 감정을 "행동"으로 묘사 (감정 단어 금지):
    * "거울 볼때마다 한숨 나옴" (O) / "신경쓰이다" (X)
    * "셀카 확대해서 모공 확인하는게 일과" (O) / "걱정됐다" (X)
  - "은근 계속 신경이 쓰이더라구요" 같은 자연스러운 심화

(3) 사회적 압박/주변 반응 (1~2줄):
  - "인스타 보면 다들 피부 좋은데 나만 칙칙"
  - "단체사진에서 내 얼굴만 칙칙해보이는거"

### 장면3-B: 스토리 후반 - 시도와 좌절 (type: "text") - 6~8줄
[감정: 걱정 → 절박 → 좌절 → 포기 직전]

★ 대본에서 가장 중요한 "감정의 바닥" (영상 40~50% 지점).
여기서 충분히 좌절해야 솔루션의 가치가 올라감.

(4) 시도와 실패 (4~5줄):
  - 최소 3가지 시도를 언급
  - 각 시도마다 "뭘 했고 + 왜 안됐는지"를 한 세트로:
    * 저가 시도: "다이소에서 ~사서 써봤는데 그때뿐"
    * 정보 탐색: "유튜브에서 ~방법 다 따라해봄" / "미백관련해서 자료도 찾아보고"
    * 민간요법: "레몬을 피부에 문지르면 하얘진다고 해서 맨날 레몬도 사서 발라봤는데 결국은 똑같더라..."
    * 고가 시도: "병원 알아봤는데 가격 보고 바로 포기ㅋㅋ"
  - 실패 경험에 구체적 기간/금액: "2주 써봤는데 변화없음", "용돈 다 날림"

(5) 전환점 (2~3줄):
  - "이렇겐 안될거 같아서 혼자 해보려고 마음먹고"
  - "그래서 걍 포기하고 있었는데"
  - "근데 인스타에서 우연히 이걸 봤거든"
  - [핵심] 전환은 반드시 "우연히/어쩌다/마음먹고" 느낌.

### 장면4: 원인/정보 (type: "image_text") - 5~7줄
[감정: 좌절 → "아 이래서 그랬구나" 납득] (영상 50~60% 지점)

왜 지금까지의 방법이 안 됐는지 원인을 설명하는 "교육" 파트.
전문적이지만 친구한테 설명하듯 쉽게. "알고보니 ~거든?" 톤.

[필수] 원인→해결 논리 구조:
  "~가 까만 이유 부터 정확히 알아야하는데"
  → "바로 피부에 **멜라닌 색소**가 많아서야"
  → "그럼 반대로 생각해서 피부가 하얘지려면??"
  → "**멜라닌 색소**가 적어지거나 더 많이 안생기면 되겠지?!??"

[필수] 기존 방법의 한계 설명:
  - "병원에선 **레이저시술**을 하는데 의사도 추천을 안하더라구요"

[정보 이미지 전략] (25개 중 20%가 사용 - 신뢰도 급상승):
  - 해부도/단면도 (유선 구조, 피부 단면, 모공 구조)
  - 성분표/비교표 (AHA/BHA/PHA/LHA 등)
  - 가격 비교표 (병원 시술 vs 홈케어)
  - 그래프/통계 (성장 곡선, 효과 데이터)

예시 1: "신경쓰이는사람 #까망이" (57초, 스토리형/1인칭/참여형)
- 썸네일: PANN "그곳이 유독 까만분 들어와주세요"
- 훅킹: 소음순 색상 투표 이미지 + "혹시 소중이가 까매서 걱정이신분?"
- 스토리(전): 한번도 신경X → 남자들 반응 듣고 → 은근 신경쓰임
- 스토리(후): 자료찾기 → 병원 알아봄 → 수백만원 써도 못지움
- 원인: 멜라닌 색소 → 레이저는 의사도 비추 → 가격표 이미지
- 솔루션: 착색전용크림 → 멜라닌 차단+억제
- 후기: 한달후 복원 + 솔직고백("처음엔 별 차이 없었는데") + 친구 반응
- 마무리: 추천
