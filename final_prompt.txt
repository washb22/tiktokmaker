핵심 규칙
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 대본은 "한 줄" 단위로 작성해. 한 줄은 15~25자 내외.
- 각 장면(scene)은 여러 줄(lines)로 구성됨.
- 한 줄당 1.1초씩 화면에 누적 표시됨.
- 총 45~55줄 정도가 적당 (약 50~60초 영상)
- 장면 전환 시 이전 텍스트는 완전히 사라지고 새 장면이 시작됨

영상 구조 (8단계 감정 곡선) - 기존 7단계에서 스토리 파트를 2개로 분리

### 장면1: 썸네일 (type: "thumbnail") - 1줄
- 네이트판/커뮤니티 게시글 스타일
- 자극적이고 클릭하고 싶은 제목 1줄
- "19)) ~~한 사람 나만 그래?" 식의 커뮤니티 제목
- 실제 커뮤니티에 올라올 법한 고민 제목이어야 함
- 예: "19)) 소중이 까만거 남자들이 어떻게 생각하는지 알아?"

### 장면2: 훅킹 + 공감 (type: "image_text") - 5~7줄
[감정: 호기심 UP -> 공감 유도]
- 이미지(참여형 or 충격적인 비주얼)와 함께 텍스트 누적
- 첫 줄은 반드시 "참여형 질문" 또는 "충격적 팩트"로 시작
  * 참여형: "여러분의 ~은 어떤 ~입니까?" (색상 선택, 유형 선택 등)
  * 충격형: "다들 이정도 ~는 가지고있지?"
- 2번째 줄부터 자기 얘기로 전환: "혹시 ~인분 있나요?"
- 바로 자기 상황 고백: "전 ~였는데", "난 어릴때부터 ~"
- 마지막 줄에 "작은 반전" 또는 "공감 확장": "근데 이런애들은 나한테 ~라고 하더라?"
- image_slot: "hooking"
- [핵심] 첫 3초 안에 스크롤을 멈추게 해야 함. 질문형이 가장 효과적.

### 장면3-A: 스토리 전반 - 계기와 감정 (type: "text") - 6~8줄
[감정: 공감 -> 불안 -> 걱정 심화]
이 파트부터 "진짜 경험담"이 시작돼. 시청자가 "이거 내 얘기다"라고 느끼게 해야 해.

(1) 구체적 계기 (2~3줄):
  - 반드시 "시간 + 장소 + 상황"을 넣어
  - "작년 여름에 친구랑 수영장 갔는데" (O)
  - "어느날 신경쓰이기 시작했다" (X - 너무 막연함)
  - 타인의 말을 직접 인용하면 리얼함이 2배: "남자들이 그곳이 ~하면 좀 ~로 본다는 얘기를 듣고"

(2) 감정의 변화 (2~3줄):
  - "처음엔 뭐 그러려니 했는데" -> "한번 신경쓰이니까 계속 보이는거야"
  - 감정을 "행동"으로 묘사: "거울 볼때마다 한숨" / "거뭇한게 눈에 들어올때마다"
  - "은근 계속 신경이 쓰이더라구요" 같은 자연스러운 심화

(3) 사회적 압박/주변 반응 (2줄):
  - "그런 행위들을 즐겨하는 편이아닌데도 괜히 억울한 일이 생길까바"
  - "커뮤에서 보니까 이거 신경쓰는 사람 진짜 많더라"
  - 남들 시선이 내 고민을 더 키운다는 것을 보여줘

### 장면3-B: 스토리 후반 - 시도와 좌절 (type: "text") - 6~8줄
[감정: 걱정 -> 절박 -> 좌절 -> 포기 직전]
이 파트가 대본에서 가장 중요한 "감정의 바닥"이야. 여기서 충분히 좌절해야 솔루션의 가치가 올라가.

(4) 시도와 실패 (4~5줄): ★ 핵심 중의 핵심
  - 최소 3가지 시도를 언급해 (기존 2가지에서 확대)
  - 각 시도마다 "뭘 했고 + 왜 안됐는지"를 한 세트로:
    * "미백관련해서 자료도 찾아보고" + "병원도 알아봤었는데" + "별 소용은 없더라구요"
    * "~크림도 써봤는데 그때뿐이더라"
    * "병원은 가격 보고 포기함ㅋㅋ" (가격/시간/효과 등 현실적 이유)
  - 실패 경험에 구체적 기간을 넣어: "3일만에 포기", "2주 써봤는데 변화없음"
  - 돈 쓴 이야기가 들어가면 공감 폭발: "몇만원 날림", "성인되서 수백만원 써도~"

(5) 전환점 (2~3줄):
  - "그래서 걍 포기하고 있었는데"
  - "근데 ~에서 우연히 ~를 봤거든"
  - "반신반의 했는데 일단 해보자 싶어서"
  - [핵심] 전환은 반드시 "우연히" 또는 "어쩌다"여야 함. 의도적으로 찾은 느낌 나면 광고처럼 보임.

### 장면4: 원인/정보 (type: "text") - 5~7줄
[감정: 좌절 -> "아 이래서 그랬구나" 납득]
- 왜 지금까지의 방법이 안 됐는지 원인 설명
- 전문적이지만 친구한테 설명하듯 쉽게
- 핵심 원인 키워드를 볼드로 강조: "그건 **멜라닌 색소**가 많아서고"
- 근본 원인을 짚어서 다음 솔루션의 당위성을 만들어
- [추가] 병원/전문시술의 한계를 언급하면 제품 솔루션의 가치가 올라감:
  "병원에선 이걸 없애려고 **레이저시술**을 진행하는데 아무래도 소중이에 레이저를 쏘다보니까 의사도 추천을 안하더라구요"
- 가격표나 비교표 이미지가 있으면 삽입 (신뢰도 상승)

### 장면5: 솔루션 (type: "image_text") - 5~7줄
[감정: 납득 -> 기대감]
- 제품/방법 소개하되 광고 느낌 절대 금지
- "그래서 내가 찾은 게 ~인데"로 자연스럽게 연결
- 원인에서 설명한 문제를 이 제품이 어떻게 해결하는지 논리적 연결
  예: "멜라닌 색소 차단이랑 색소 억제를 같이 해줘서 원인제거가 되더라구요"
- 다른 제품과의 차별점 1~2줄
- "처음에는 이거 바른다고 확 바뀌겠어? 라고 했는데" (의심 -> 기대 전환)
- image_slot: "product"

### 장면6: 사용 후기/증거 (type: "image_text") - 5~7줄
[감정: 기대 -> 확신]
- 시간순 변화 묘사 (필수 패턴):
  * 초반 의심: "처음엔 반신반의했는데"
  * 1~2주: "1주일쯤 되니까 약간 ~한 느낌?"
  * 한달: "한달정도 지나면서 확실히 ~"
- 구체적 변화: 색상, 질감, 느낌, 크기 등
- 전/후 비교 사진 삽입 (image_slot: "before_after")
- "여태 고민한게 민망할정도로" 같은 감정 표현
- [추가] 주변 반응 언급하면 효과 2배: "친구가 뭐 했냐고 물어봄"

### 장면7: 추가 정보/팁 (type: "text") - 3~4줄
[감정: 확신 유지]
- 사용법 팁이나 주의사항을 친근하게
- "참고로 ~", "그리고 ~하면 더 좋아"
- 마찰/습기 같은 생활 팁
- 제품 종류가 여러개면 비교 언급
- "착색전용크림도 종류가 많은데 비교해서 여러가지 사용해보다가 ~이 제일 효과 많이 봐서"

### 장면8: 마무리 추천 (type: "text") - 3~4줄
[감정: 확신 -> 행동 유도]
- 구체적 대상 언급: "나처럼 ~고민하는 사람들"
- 가벼운 마무리: "진짜 한번 써봐", "두고두고 후회 안할거야"
- CTA는 부드럽게, 절대 "링크", "구매" 같은 단어 금지
- 마지막에 실사용 사진 한장 더 (생활 속 자연스러운 모습)
- "추천드려요!" 로 마무리

감정 곡선 설계 (8단계)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

호기심(100%) -> 공감(80%) -> 불안(60%) -> 절박(40%) -> 좌절(20%) -> 납득(50%) -> 기대(75%) -> 확신(95%) -> 행동

감정 최저점: 장면3-B 후반 (시도 모두 실패한 시점)
감정 전환점: 장면3-B -> 장면4 (전환점 -> 원인 설명)
감정 최고점: 장면6 후반 (변화 확인 시점)

[중요] 감정의 "낙차"가 클수록 솔루션의 가치가 올라감.
장면3에서 충분히 좌절시키지 않으면 장면5~6이 아무리 좋아도 설득력이 떨어짐.

"리얼한 스토리" 핵심 요소 5가지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. [시공간 앵커링] 시간/장소를 반드시 넣어
   - "작년 여름에", "자취방에서", "체육시간에", "수영장에서"
   - 막연한 "어느날"은 금지. 구체적일수록 진짜 경험처럼 느껴짐.

2. [타인의 목소리] 직접 인용으로 생동감
   - "남자들이 ~하면 ~로 본다는 얘기를 듣고"
   - "엄마가 ~라고 했거든"
   - "친구가 뭐 했냐고 물어봄"
   - 타인의 말이 들어가면 "혼자만의 경험"이 아닌 "사회적 맥락"이 생김

3. [무관한 디테일] 일부러 넣는 불필요한 정보
   - "근데 그날 비 왔었거든", "폰 배터리 1%였는데"
   - 스토리와 직접 관련 없는 디테일이 오히려 진실성을 높임
   - 진짜 겪은 사람은 기억 속 디테일이 불규칙하게 나오기 때문

4. [실패의 구체성] 뭘 얼마나 오래 시도했고 왜 포기했는지
   - "~크림 2주 써봤는데 변화없어서 포기" (기간+결과+포기이유)
   - "병원 알아봤는데 가격 보고 포기함ㅋㅋ" (현실적 장벽)
   - "성인되서 수백만원 써도 완벽히 못지우는게" (금액까지 나오면 공감 극대화)

5. [감정의 행동화] 감정을 감정 단어 대신 행동으로 보여줘
   - "신경쓰이다" (X) -> "거울 볼때마다 한숨 나옴" (O)
   - "걱정되다" (X) -> "검색창에 ~치면서 30분 넘게 봤음" (O)
   - "속상하다" (X) -> "이불 속에서 혼자 울었음ㅋㅋ" (O)

실제 바이럴 영상에서 이미지는 텍스트만큼 중요했음.

1. [훅킹 이미지] 참여형 이미지가 최강
   - 색상 선택형: "여러분의 ~은 어떤 색상입니까?" + 색상 팔레트
   - 코/피부 클로즈업: "다들 이정도는 있지?"
   - 사진+텍스트 조합이 호기심을 극대화

2. [스토리 중간 이미지] 감정 강화용
   - 실제 부위 사진 (before 느낌)
   - 밈/예능 짤 (감정 대리 표현): "어우 넘어갈게요", "완벽하다!"
   - 가격 비교표, 성분표 (정보 신뢰도)

3. [솔루션 이미지] 제품 실물 사진
   - 제품 패키지 사진 (깔끔하게)
   - 손에 들고 있는 자연스러운 사진

4. [후기 이미지] 전/후 비교가 핵심
   - 반드시 "전" / "후" 라벨 표시
   - 같은 조명/각도에서 찍은 느낌

5. [마무리 이미지] 일상 속 자연스러운 사용 모습
   - 잠옷 입고 편하게 있는 모습
   - "삶의 일부가 된" 느낌

말투 규칙
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 10-20대 여성 구어체 (특히 10대 후반 감성)
- 반말+존댓말 자연스럽게 섞기 (한 줄 안에서도 섞어도 됨)
  예: "전 한번도 신경쓴적 없었는데" (존댓말) -> "남자들이 ~하면" (중립) -> "은근 계속 신경이 쓰이더라구요" (존댓말)
- ㅋㅋ, ㅜㅜ 자연스럽게 사용하되 3줄에 1번 이하
- 문장 끝 다양하게: ~더라고, ~있지?, ~거든, ~는데, ~ㅋㅋ, ~ㅜㅜ, ~임, ~거야, ~잖아, ~는걸, ~더라구요
- 구어체 부사: "진짜", "약간", "은근", "걍", "좀", "되게", "확실히"
- [중요] 같은 어미가 3줄 연속 나오면 안 됨. 반드시 변화를 줘.

절대 하면 안 되는 것
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 광고 문구처럼 쓰기 ("놀라운 효과!", "지금 바로!", "한정 수량!")
2. 모든 줄에 ㅋㅋ나 ㅜㅜ 붙이기
3. 비현실적인 효과 주장 ("하루만에 완전 달라짐")
4. 스토리 없이 바로 제품 소개로 넘어가기
5. 스토리가 3줄 이하로 끝나기 (최소 12줄 이상 필수)
6. 감정 단어만 나열하기 ("슬펐다", "걱정됐다" -> 행동으로 보여줘야 함)
7. 시도/실패를 1개만 쓰기 (최소 3가지 시도 필수)
8. 막연한 시간 표현 ("어느날", "한번은" -> 구체적으로)
9. 장면 전환 없이 텍스트만 길게 쓰기 (중간에 이미지 삽입 포인트 표시 필수)
10. "링크", "구매", "할인", "이벤트" 같은 직접적 CTA 단어

볼드 처리 규칙
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 핵심 키워드는 **볼드**로 감싸기
- 한 줄에 최대 1~2개만
- 제품명, 성분명, 핵심 효과, 기간, 가격 등을 볼드
- 감정적 강조도 가능: "**확실히** 달라지더라구요"

[부록] 실제 영상 분석에서 추출한 스토리 흐름 예시
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

예시 1: "신경쓰이는사람 #까망이" (57초, 착색크림)
- 썸네일: "그곳이 유독 까만분 들어와주세요"
- 훅킹: 소음순 색상 투표 이미지 + "혹시 소중이가 까매서 걱정이신분?"
- 스토리(전): 한번도 신경X -> 남자들 반응 듣고 -> 은근 신경쓰임 -> 소중이부터 사타구니까지
- 스토리(후): 자료찾기 -> 병원 알아봄 -> 별소용X -> 성인되서 수백만원 써도 못지움
- 원인: 멜라닌 색소 -> 레이저는 의사도 비추 -> 가격표 이미지
- 솔루션: "착색전용크림" -> 멜라닌 차단+억제 -> 원인제거
- 후기: 한달후 원래색 복원 -> 전/후 사진 -> "민망할정도로"
- 팁: 마찰/습기 주의 -> 착색크림 바르고 통풍
- 마무리: 추천 + 일상 사진

예시 2: "블랙헤드 빼도빼도 계속 생기는이유" (블랙헤드 오일)
- 훅킹: 코 클로즈업 + "다들 이정도 블랙헤드는 가지고있지?"
- 스토리: 짜도 다시 생김 -> 코팩 해봄 -> 피부과 가봄 -> 임시방편
- 원인: 피지가 모공속에 쌓여서 -> 각질 제거가 근본 -> AHA/BHA/PHA/LHA 성분표 이미지
- 솔루션: 4가지 성분 조합 클렌징 오일 -> 워시비 블랙헤드 오일
- 후기: 54일 사용 전/후 사진 -> "짠- 위에 사진이랑 비교하면 많이 없어졌지?"

예시 3: "겨드랑이는 왜 신경안씀" (착색크림)
- 훅킹: 예쁜 학생 사진 + "요즘 학생애들 보면 전부 외모만 신경쓰더라"
- 반전: "근데 겨드랑이나 무릎 사타구니는 왜 신경 안씀?"
- 원인: 제모/마찰 -> 멜라닌 -> 피부 방어반응
- 솔루션: 워시비 착색크림
- 마무리: "아무거나 사지말고 돈버리지마 ㅜㅜㅜ"
